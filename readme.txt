runme.m: Run this to use the application
findTopMatches.m: Finds top matches in given database

binarize.m: Binarize an image, return the mask

getColorFeatures.m: Calculate color descriptors for an image
compareColorFeatures.m: Calculate similarity score between two color descriptors
sRGB2XYZ.m: Converts RGB to XYZ space
deltaE00.m: Calculate CEIDE2000 color difference

gaborcoefficients: Calculate Gabor coefficients for an image
createGaborFilters.m: Generate the family of Gabor filters, store in gaborfilters.mat
gaborfilter.mat: Structure containing family of Gabor filters

featureDetection.m: Calculates feature descriptors for an image using VLFeat
getFeatureScore.m: Calculate similarity score between two feature descriptors
AffinePairwiseRansac.m: Runs RANSAC algorithm to remove non-affine transformations in feature matches

fourDesc.m: Calculate Fourier Descriptors for an image
